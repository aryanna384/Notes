| 单位 | 描述                                                         |
| ---- | ------------------------------------------------------------ |
| em   | 相对单位，相对于父元素的字体大小（不推荐使用）               |
| px   | 像素单位                                                     |
| rem  | 相对基础字体单位的缩放倍数                                   |
| vw   | 相对于视窗宽度的百分比`10vw` 的意思是视窗宽度的 10%          |
| vh   | 相对于视窗高度的百分比                                       |
| vmin | 相对于视窗宽度/高度较小的值的百分比 `70vmin` 的意思是视窗的高度和宽度中较小一个的 70% |
| vmax | 相对于视窗宽度/高度较大的值的百分比                          |
| dp   | 像素密度，不同手机不一样                                     |
| fr   | 栅格布局特定单位，用来定义网格轨道大小，按比例分配剩余的可用空间 |
